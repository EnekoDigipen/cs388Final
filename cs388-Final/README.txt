resources:
 - para entrar a niveles, hay que gastar cierta cantidad de energia
 - cada lanzamiento, gastas cierta cantidad de bolas. Si te quedas sin, ajo y agua y ahi te quedas ;) O haces que los 
   players gasten para seguir

shop in-game:
 - skins
 - power up in-game:
   - magnet (3 per day)
   - x2 coins (3 per day)
   - shield (for hazards) (3 per day)

shop Pay2Play:
 - more coins
 - power up paying:
   - more time
   - more balls per level
   - more energy
   - more magnet
   - more x2 coins
   - more shields

rewards:
 - coins between 20 - 150 (80%)
 - in-game power ups (5%)
 - keys (to unlock endless mode) (14%)
 - skins (1%)

game mode:
 - normal
 - endless
 - casino

mechanics:
 - normal blocks
 - breakable block (rebota una vez y se rompen)
 - portals
 - hazard blocks
